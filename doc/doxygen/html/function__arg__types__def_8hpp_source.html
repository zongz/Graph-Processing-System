34 /*
35 A huge collection of useful typedefs.
36 F0... F5: identifies the arguments for an RPC aware function F. Dropping const and dropping references
37  (therefore allowing you to use F0....F5 to do casting.
38 
39 __GLRPC_NIF0... __GLRPC_NIF5: identifies the arguments for an RPC unaware function F
40 
41 __GLRPC_R0.... __GLRPC_R7: Identifies the actual arguments of the function F, without de-consting and de-reffing
42 
43 __GLRPC_FRESULT: de-const and de-refed type of F's return type
44 
45 __GLRPC_FARITY: the number of arguments F takes
46 */
47 #define REMOVE_CONST_REF(REF) typename boost::remove_const<typename boost::remove_reference<REF>::type>::type
