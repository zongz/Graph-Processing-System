ZK_SERVERS: A comma separated list of zookeeper servers. Port number must be included. ZK_JOBNAME: The name of the job to use. This must be unique to the cluster. i.e. no other job with the same name must run at the same time ZK_NUMNODES: The number of processes to wait for
